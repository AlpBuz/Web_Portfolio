Color Game
Personal Project, Summer 2024
Tools and Skills:
  • React: Used for building the dynamic and interactive user interface.
  • Python (Flask): Powered the backend server to handle game logic and data management.
  • Daisy UI (Tailwind): Implemented for streamlined and responsive UI components.

Revitalizing an old project, I focused on improving user interaction using what I learned with the POS system project. This version of the Color Game involves a quick reflex test where players must type the name of the color they see on the screen, not the color the word spells out. For instance, if the word “red” is colored blue, the correct answer is “blue”, not “red”.

  • Starting Page: A welcoming and intuitive landing page that sets the stage for the game.
  • Gameplay: An interface where users type the correct color under a time constraint
  • Leaderboard: For users to compete with friends to get the highest score
